The folder "old" contains some miscellaneous items. It is not necessary for the current method.

"calculate_input_normalisation.py" is used to calculate the standard deviation and mean of input normalisation.

"tools_sceneflow.py" is used to pre-process the sceneflow dataset,
including conversion to the appropriate format and scaling into training and test sets.

"visual_feature_map.py" is used to print the middle feature map of the network,
which can be inserted into the folder "network" in the folder "MA" as required,
in order to observe the middle feature map.
